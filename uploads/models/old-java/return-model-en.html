Java applet below solves differential equations above using Euler-Maruyama method [cite id="Kloeden1999Springer"]. We also use variable time step. In such case stochastic differential equations becomes a set of difference equations for time and return. In case of the last stochastic differential above \begin{equation} x\_{i+1} = x\_{i} + \kappa^2 \left \[ \eta -\frac{\lambda}{2} - \left( \frac{x\_i}{x\_{max}} \right)^2\right\] x\_i + \kappa \sqrt{1+x\_i^2} \xi\_i , \end{equation} \begin{equation} t\_{i+1} = t\_{i} + \kappa^2 \frac{\left( 1 + \epsilon\sqrt{1+x\_i^2} \right)^2}{\left( 1+x\_i^2 \right)^{\eta-1}}, \end{equation}

here \( \xi_i \) is Gaussian noise (zero mean, unit variance) and \( \kappa \) is numerical precision parameter.
